Write-up

Intro

This dataset was taken from Chicago Police Departments's CLEAR program. In this database, there are all instances of crime reported in Chicago from 2001 to the present day. For each report, we looked into the year and type of crime committed.

Design Decisions

We allow the user to highlight each line (type) on the graph to see more info on how many reports were there on that year. We also allow the user to filter certain types of crimes by using checkboxes at the bottom. We ended up picking a line graph as we it felt like the standard for comparing data over time. Line graphs would also make it easy for the user to compare trends. We considered taking other subsets of data and visualize using bar charts. One approach we thought of was comparing districts and crime rates. We ultimately decided to pick a line graph since we wanted to show trends over time.
